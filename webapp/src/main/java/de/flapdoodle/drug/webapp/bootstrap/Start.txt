Drug
===

Das alles ist noch ein Test, eine Idee, eine Vision. Aber es fühlt sich jetzt schon viel besser an.

Aufbau
----

Neben der Beschreibung von [[Begriffen]] wird die Beschreibung von Relationen ermöglicht. Dabei wird eine System verwendet, dass eine Adressierung nach [[Subjekt]],[[Prädikat]] und [[Objekt]] vornimmt. Das hat nette Nebeneffekte, wie das [s:->Mensch] [p:filtern] [o:->Drug] nach verschieden Relationsbestandteilen.

Nächste Schritte
---
Das ganze muss man noch um einen Kontext erweitern. Der Kontext kann etwas beliebiges, aber Optional sein. Beispiele:

* [s:->Mensch] mit dem [o:Zug] [p:fahren]
* [s:->Mensch] mit dem [o:Zug] [p:fahren] in der [at:Schweiz]
* [s:->Mensch] mit dem [o:Zug] [p:fahren] in [at:Niedersachsen]

Den Kontext könnte man immer angeben mit [\@Schweiz]. Außerdem könnte man eine alternative Syntax angeben. Statt [\spo:displayText->Text] könnte man (s:Bla) (@Blub) anbieten wobei eckige Klammern wohl eher dem MarkDown-Gedanken entsprechen.
